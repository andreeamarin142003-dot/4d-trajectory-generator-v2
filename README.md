# 4d-trajectory-generator-v2
As part of ATM seminar activity
Subject 2 – 4D Trajectory Generation 
Using data from AIP, generate a valid flight route between an ADEP and ADES. Generate 
the flight profile as a sequence of trajectory samples (Lat, Long, alt, speed) for a certain 
aircraft type using a chosen aircraft performance model (variable speeds and vertical 
rates of change). Trajectory climbs ASAP to the optimum cruise level and descends ALAP. 
No wind.

HOW TO USE:
(you should have Python installed)

Open VS Code --> Terminal --> New Terminal --> pip install -r requirements.txt --> python -m streamlit run app.py
